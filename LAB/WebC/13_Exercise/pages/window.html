Window

window object คือ หน้าต่างของ browser ที่กำลังเปิดหน้าเว็บอยู่


โดย window นั้นมีหัวข้อหลักๆอยู่ 2 หัวข้อด้วยกันนั่นก็คือ window feature และ window object

  • window feature เป็นในส่วนของการจัดหน้า layout ของหน้า html โดยจะมีคำสั่งหลักๆคือ
    • moveTo(x,y) ใช้เลื่อนหน้าต่างไปยังตำแหน่งพิกัด x, y ตามที่กำหนด
    • moveBy(x,y) ใช้เลื่อนหน้าต่าง โดยจะนำค่า x, y ที่กำหนด ไปบวกลบกับตำแหน่งพิกัด x, y เดิม
  • window object เป็นในส่วนของการใช้คำสั่งต่างๆในการเเสดงหน้าเว็ปหรือตอบสนองกับ user ตัวอย่างคำสั่งมีดังนี้
    • alert() ใช้ในการสร้าง message dialog
    • confirm() ใช้ในการสร้าง question box
    • prompt() ใช้ในการสร้าง input box โดยจะคืนค่าข้อมูลที่กรอกกลับมา

Window Outer Size คือ ขนาดหน้าจอที่รวมทั้ง browser

Window Inner Size คือ ขนาดหน้าจอส่วนที่แสดง Content (รวม Scroll bar )

Window Position เป็นการย้ายตำแหน่งของ Window ไปที่ตำแหน่งหน้าจอทีเรากำหนด โดนใข้ method moveBy()

Window Resize เป็นการ เพิ่ม/ลด ขนาดของ window โดยใช้ method reziseBy()
