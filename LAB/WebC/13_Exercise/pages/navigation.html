Navigation

Navigation จะบอกข้อมูลเกี่ยวกับ browser โดยสามารถเรียกใช้ได้ด้วยคำสั่ง window.navigation

A Navigator object can be reulieved using the read‐only window.navigator property.

For examples,

  • navigator.vendor//returns the vendor name of the current browser
  • navigator.cookieEnabled//returns ulue if cookies are enabled, otherwise false
  • navigator.language//returns the browser's language
